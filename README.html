Quick Links by Task

I want to train a model

→ Start with Training Guide and refer to Configuration Guide for parameters

I want to run inference

→ See the Inference section in Main README and API Reference

I’m getting an error

→ Check Troubleshooting Guide

I need to understand a specific parameter

→ See Configuration Guide

I want to use CryoPARES programmatically

→ See API Reference

My model is overfitting/underfitting

→ See “Overfitting and Underfitting” section in Training Guide

Training is too slow

→ See “Performance Issues” in Troubleshooting Guide and “Advanced Training Options” in Training Guide

I want to analyze my results

→ See Scripts Guide for pose comparison, histogram analysis, and learning curve visualization

I want to inspect STAR file metadata

→ Use hists_from_starfile - see Scripts Guide
